核心设计理念：果味储物间 🍏
关键词： 极简、留白、轻拟物、通透、动效自然、柔和阴影。
核心隐喻： 整个 App 是一组悬浮的数字货架，商品像被轻柔托起的精致卡片，用户感受到的是“轻盈、干净、专注”的体验，而不是拥挤的储物柜。


一、UI 配色方案：G. 果味储物间 (Apple Pantry)
苹果风格强调留白和极简，所以 UI 配色主要是浅灰+白+柔和蓝，并保留红黄绿状态色用于保质期提示，但颜色饱和度更柔和，避免刺眼。
角色色值视觉描述与用途背景底色#F5F5F7柔和浅灰，带轻微渐变感，营造空间感。货架层/卡片rgba(255, 255, 255, 0.85) + Blur: 30px半透明浮层，圆角极大，轻微阴影模拟浮在背景之上。品牌主色#0A84FF (iOS 蓝)用于主要按钮、选中状态、关键操作，柔和而清晰。文本色(深)#1C1C1EApple 系统一贯深灰黑，用于标题和重要信息。文本色(浅)#3C3C4399辅助说明、二级信息，如分类名。状态色：安全#30D158 (iOS 绿)剩余时间充裕 (>30天)，柔和、易读。状态色：临期#FFD60A (iOS 黄)需要注意 (<15天)，不刺眼。状态色：紧急#FF453A (iOS 红)即将过期 (<3天)，醒目但柔和过渡。
核心变化：整体颜色柔化，玻璃卡片更轻盈，阴影细腻，保留状态色提示但更“苹果味”。


二、关键界面 UI 设计细节
1. 首页：“我的货架” (Dashboard)
布局：两列网格（iOS 风格 CollectionView），卡片间距更大，留白更多，突出商品视觉。
顶部区域 (Header)：
极简问候：Hi, 你的库存状态良好。
状态概览条：半透明圆角浮层，带轻微阴影，显示“🔴 2件急需处理 | 🟡 5件临期”，点击可筛选。
商品卡片 (Item Card)：
容器：圆角 20~30px，轻阴影 (Shadow: 0 4px 12px rgba(0,0,0,0.05))，漂浮感明显。
图片：商品图片居中，带圆角 12px，背景透明卡片。
生命进度条：细条状，颜色从绿色到红色渐变，轻微阴影表示“轻盈悬浮”。
剩余天数徽章：右上角圆角标签，半透明浮层，字体柔和。
标题与信息：底部文字居中，字体用 SF Pro 系列，统一 Apple 风格。


2. 核心交互入口：超级扫码钮
底部悬浮大按钮：
圆形，深蓝 (#0A84FF)。
微微浮起（Shadow + Scale Effect）。
动效：轻柔呼吸，不抢眼但引导注意。
图标：简洁扫描框 + 闪电，保持 Apple 风格线条简约。


3. 扫码录入流程
扫描界面：全屏相机，取景框圆角，四角轻微高光蓝色，背景模糊。
识别反馈：
成功识别：取景框轻微收缩 + 绿色淡光闪烁 + 手机震动。
底部浮层：半屏透明卡片，左侧缩略图，右侧自动填充信息。
“确认入库”按钮：圆角大按钮，iOS 蓝，微微阴影。


4. 空状态 (Empty State)
画面：极简浮层储物架，几件小物漂浮，背景留白。
文案：你的数字储物间空空如也。
引导：弯曲箭头指向底部扫码按钮，保持留白和视觉呼吸感。


三、关键交互体验
视差滚动：卡片层和背景层轻微差速滚动，3D 浮动感。
临期物品：红色卡片微微呼吸闪烁，柔和而不刺眼。
消耗操作：卡片缩小飞向底部回收图标，动画柔和，带轻微音效（盖子轻合声）。
